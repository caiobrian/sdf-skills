# Tasks Template

```markdown
# Tasks: [name]

## T-01: [descriptive name]
- **Type:** component | hook | util | service | test | config
- **Requirements:** RF-01, RNF-02
- **Input:** [precondition]
- **Output:** [deliverable]
- **Done:** [objective criteria]
- **~Lines:** [estimate]
- [ ] Pending

## T-02: ...
```

Each task references requirement IDs. Max ~50 lines per task. Ordered by dependency. One responsibility per task — split if it does two things.
